# 鼠鼠天堂 Hamster Paradise

SillyTavern 同层前端角色卡 — 经营一座梦幻般的宠物鼠乐园，收养被遗弃的鼠鼠，与拥有超自然能力的鼠天使一起守护它们。

## 简介

一款基于 SillyTavern 的轻量经营游戏角色卡。玩家经营一座收养被抛弃宠物鼠的乐园，风格温馨治愈。

- **玩家**：乐园的主人
- **鼠天使**：拥有种种超自然能力的管理员工
- **鼠居民**：被收养的普通鼠鼠，不需要工作，只需要玩耍 — 而玩耍会发电供能，能源量是乐园的底层货币

## 依赖

使用本角色卡需要安装以下 SillyTavern 扩展：

- [酒馆助手](https://github.com/StageDog/tavern_helper)
- [提示词模板](https://github.com/zonde306/ST-Prompt-Template)
