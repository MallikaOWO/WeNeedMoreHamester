## 发现的问题与修复

### Bug 1: `generate()` 不能正确绕过预设（已修复）
- **问题**: `summarizeAndHide` 使用 `generate()` + `overrides` 试图绕过世界书和角色描述，但预设中的系统提示词（叙事角色设定）仍然被发送，导致 AI 生成游戏叙事而非总结
- **修复**: 改用 `generateRaw()` + `ordered_prompts`，完全绕过预设，只发送总结提示词
- **文件**: `src/鼠鼠天堂/界面/主界面/store.tsx` 第 397 行

### Bug 2: `setChatMessages` 的 `refresh: 'none'` 不持久化隐藏（已修复）
- **问题**: 使用 `refresh: 'none'` 调用 `setChatMessages` 设置 `is_hidden: true` 时，更改不生效
- **修复**: 改为 `refresh: 'affected'`
- **文件**: `src/鼠鼠天堂/界面/主界面/store.tsx` 第 432 行

### Bug 3: `generateRaw` 需要 `should_silence: true`
- **问题**: 不加 `should_silence` 时 `generateRaw` 会产生临时消息和停止按钮交互
- **解决**: 加入 `should_silence: true`，确认不产生临时消息，无需删除步骤
