提示词模板基础用法：
https://github.com/zonde306/ST-Prompt-Template/blob/main/README_CN.md
我们只需要基础用法即可，不需要复杂用法，其它功能均有mvu原生功能实现，但我们借助此逻辑实现全蓝灯世界书情况下，根据注入的变量不同实现动态发送提示词：
```
条件逻辑 根据变量的值改变提示词内容:
<% if (getvar('好感度', { defaults: 0 }) > 50) { %>
你是我信赖的朋友。
<% } else { %>
我仍然对你保持警惕。
<% } %>
```
